# LinkSaaS – Link Management Platform

A simple SaaS-style link management app, like Linktree, built with:

- 🔧 Backend: Express.js + PostgreSQL
- 🎨 Frontend: Next.js
- 🧪 Auth: JWT Token
- 🛡️ Role: Private (per-user dashboard)
- 🚀 Status: Active development

## 📂 Project Structure

my-linksaas/
├── backend/ → Express.js API (JWT, PostgreSQL)
├── frontend/ → Next.js App (dashboard, link preview)
├── .env.example → Example environment config
---

## 📦 Tech Stack

- Node.js (Express)
- PostgreSQL
- JWT
- Next.js
- React

## 🚀 How to Run

### 🔧 Frontend &  Backend
```bash
cd frontend
npm install
npm run dev

### 🔧 Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
